Variable and data types

To declare variables in Python, you assign a value to an identifier with the assignment (=) operator. 
No need for keywords

e.g:
name = 'John Doe'
age = 25

-Should start with letter or underscore
-Contains alphanumeric characters and underscores
-They are case sensitive
-Dont use preserved python keywords
-They should be in lower casing
-They should be descriptive
-No single character variable names

Common Data Types in Python and How Do You Get the Type of a Variable

A data type describes the kind of value a variable holds. 
The language knows what data type a variable is based on what you assign to it.

example=name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer

With python , code is checked while it is running, not when it is compling.
 Compiling means the computer checks your code in advance and prepares it to run

Because of this, you might not learn about a type mistake in Python until the program reaches that specific line of code while running.

COMMON DATA TYPES:
Integer:my_integer_var = 10

Float:my_float_var = 4.50

String:my_string_var = 'hello'

Boolean:my_boolean_var = True

Set:my_set_var = {7, 5, 8}=An unordered collection of unique elements

Dictionary=A collection of key-value pairs enclosed in curly braces
my_dictionary_var = {'name': 'Alice', 'age': 25}

Tulpe=An immutable ordered collection, enclosed in brackets
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var)
# Tuple: (7, 5, 8)

Range:A sequence of numbers, often used in loops
my_range_var = range(5)
print('Range:', my_range_var)
# Range: range(0, 5)

List=An ordered collection of elements that supports different data types.
my_list = [22, 'Hello world', 3.14, True]

None type= my_none_var = None

To get data type of the variable:

my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>


The built-in isinstance() function lets you check if a variable matches a specific data type. 

isinstance(object, type)
returns a boolean

example=
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
